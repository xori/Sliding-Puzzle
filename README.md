## Sliding Puzzle

### How to use.
```
./COMPILE
./RUN
```
or

```
./COMPILE
./RUN [[STARTING STATE] [GOALSTATE]]
```

### Example
```./RUN 3 5 6 7 4 2 1 0 8```
or
```./RUN 3 5 6 7 4 2 1 0 8 8 7 6 5 4 3 2 1 0```
